# Youdemy – Online Educational Platform

Youdemy is a full-stack online learning platform that enables students to explore courses, enroll in programs, and track their learning progress. The platform provides a seamless educational experience for learners and an efficient course management system for administrators.

---

## 🚀 Features

### 👨‍🎓 Student Features
- User registration and login
- Browse and search available courses
- View course details and descriptions
- Enroll in courses
- Track learning progress


### 👨‍💼 Admin Features
- Secure admin login
- Add, update, and delete courses
- Manage student enrollments
- Monitor platform usage
- Organize courses by category


## 📸 Key Modules

### Authentication Module
Handles registration, login, and role-based access control.

### Course Management Module
Allows admins to manage courses and categories.

### Enrollment Module
Enables students to enroll in courses.

### Progress Tracking Module
Tracks completed lessons and quiz performance.
